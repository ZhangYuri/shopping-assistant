## 接口 `https://yuri.iqiyi.coms/`

|名称|path|method|body|headers|response|
|:---|:---|:---|:---|:---|:---|
|订单导入|/order/import|POST|{file: File[], channel: string}|content-type: multipart/form-data|200 ok|
|库存录入|/inventory/insert|POST|{order_ids?: string[]} <br/> {name: string, file: File, quantity: number}|content-type: application/json|200 ok|
|库存修改|/inventory/update|PUT|{id: string, file?: File,current_quantity?: number }|content-type: multipart/form-data|200 ok|
|购物清单列表|/shopping/list|GET|/|/|body: {item_name: string, suggested_quantity: number,priority: number, status: string,reason: string}[]|
|购物清新增|/shopping/insert|POST|{item_name: string, suggested_quantity: number,priority: number, reason: string}|content-type: application/json|200 ok|
|购物清更新|/shopping/update|PUT|{id: number,item_name: string, suggested_quantity: number,priority: number, reason: string}|content-type: application/json|200 ok|
|购物清移除|/shopping/remove|DELETE|{id: number}|content-type: application/json|200 ok|

## 数据库

### 购物历史清单
|字段|类型|是否必填|备注|
|:---|:---|:---|:---|
|id|string|Y|订单编号|
|store_name|string|Y|店铺名称|
|total_price|decimal|Y|总价
|delivery_cost|decimal|Y|运费
|pay_fee|decimal|Y|实付金额
|purchase_date|timestamp|Y|下单时间
|purchase_channel|string|Y|下单平台
|created_at|timestamp|Y|自动填充|

### 购物历史子清单
|字段|类型|是否必填|备注|
|:---|:---|:---|:---|
|id|bigint|Y|自增|
|parent_id|string|Y|父订单id|
|category|string|Y|/|
|item_name|string|Y|商品名称
|model|string|Y|规格|
|purchase_quantity|int|Y|商品数量
|unit_price|decimal|Y|单价
|created_at|timestamp|Y|自动填充|

### 库存表
|字段|类型|是否必填|备注|
